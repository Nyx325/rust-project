# Acerca de
Este proyecto plantea la estructura de un sistema en rust que debe 
gestionar mediante sqlite algunos datos, no tiene mucho misterio

## Como ejecutar
Descargar rust de la pagina oficial y el gestor de paquetes
cargo

Ejecutar
```bash
cargo run
```
